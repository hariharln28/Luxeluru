# Luxeluru

**Bengaluru's Premier Luxury Salon Finder**

A responsive web app for discovering, booking, and navigating luxury beauty salons across Bengaluru.

## Features

- **Auth** — Register & login (no OTP), persistent sessions
- **15 Luxury Salons** — Services, packages, pricing across Bengaluru
- **AI Style Advisor** — Camera-based face tone/shape analysis with hair colour suggestions
- **KNN Salon Navigator** — Proximity-ranked salon finder with map integration
- **Booking** — Date/time/stylist selection, Cash/UPI at salon
- **Notifications** — WhatsApp & email confirmation on booking; 24h feedback via WhatsApp
- **Leaderboard** — Top salons & stylists with customer ratings
- **Appointment History** — Full booking history for logged-in users
- **Panic Button** — Instant nearest salon finder on dashboard
- **Salon Partner Portal** — 5% monthly commission with grace period
- **i18n** — English (default), Hindi, Kannada
